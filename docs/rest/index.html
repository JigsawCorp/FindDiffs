Create a new user on the server

  • Request
  • Response

Body

Media type: multipart/form-data

Type: object

Properties

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
  "id":"438943af323ed43",
  "username":"my user"
}

HTTP status code 409

There is already a user with the same username on the server

Body

Media type: application/json

Type: object

Example:

{
  "title":"Error",
  "message":"There is already a user with the same username on the server"
}

/image-pair

Pair of original and modified images

/image-pair post

post

Create a new image-pair with new differences

post /image-pair

Create a new image-pair with new differences

  • Request
  • Response

Body

Media type: multipart/form-data

Type: object

Properties
  • name: required (string)
  • originalImage: required (file)
  • modifiedImage: required (file)

HTTP status code 200

The image pair was created successfully

Body

Media type: application/json

Type: object

Example:

{
  "id":"438943af323ed43",
  "url_difference":"http://localhost:3000/image-pair/438943af323ed43/difference",
  "url_modified":"http://localhost:3000/image-pair/438943af323ed43/modified",
  "url_original":"http://localhost:3000/image-pair/438943af323ed43/original",
  "name":"The name of the pair",
  "creation_date":"2019-01-27T19:50:48.036Z",
  "differences_count":7
}

HTTP status code 400

There is an error with the request

Body

Media type: application/json

Type: object

Example:

{
  "title":"Error",
  "message":"The field originalImage is not present in the request"
}

post /gamecard

  • Request
  • Response

Body

Media type: multipart/form-data

Type: object

Properties
  • name: required (string)
  • image-pair-id: required (string)
  • pov: required (one of Simple, Free)

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

"id":"432423423423ef",
"pov":"Simple",
"title":"The name of the gamecard",

  "id":"438943af323ed43",

  "url_modified":"http://localhost:3000/image-pair/438943af323ed43/modified",
  "url_original":"http://localhost:3000/image-pair/438943af323ed43/original",
  "name":"The name of the pair",
  "creation_date":"2019-01-27T19:50:48.036Z",
  "differences_count":7
  },
"best_time_solo":[1254512,1282445,544512],
"best_time_online":[1254512,1282445,544512]
}

Body

Type: object
